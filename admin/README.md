# Panel CROCHETGLAM
Una página minimalista para acceso directo a:
- Cloudinary (subir imágenes)
- Google Sheet (editar productos)

## Configuración
1) Sube `panel.html` y `assets/` junto a `index.html`.
2) En `panel.html`, edita:
   const PANEL_LINKS = {
     cloudinary: 'https://console.cloudinary.com/console/c-TU_CLOUD/media_library',
     sheet: 'https://docs.google.com/spreadsheets/d/TU_SHEET_ID/edit'
   };
3) Reemplaza `assets/logo-placeholder.svg` por el logo real si quieres.
